時間及び金額にまつわるドメインモデル群を含むクラスライブラリ。

オリジナルはTime & Money Libraryであり、 Eric Evansらによるオープンソースプロダクト（MIT License）である。 このプロジェクトは非常に有意義なDDD(Domain Driven Design)の実例である以前に、Javaにおける時間及び金額にまつわる プログラミングに大きく貢献するクラスライブラリである。 しかし残念なことに、Domain Language, Inc.のページを見るところによると 「このプロジェクトはもはやアクティブではない(Although this project is no longer active)」と明記している。

従って、本プロジェクト（Time & Money Remodelled Library）は、Time & Money Libraryにおける 不完全（主に仕様ドキュメンテーション）な部分を補い、個人的な設計ポリシー（例外の扱い方や、Comparatorの提供など）を適用し、 いくつかの僅かなバグを修正したものを、派生プロダクト（Apache License v2.0）としてリリースするものである。

Eric Evans has been the primary developer and designer of TimeLanguage. Martin Fowler contributed the initial version of Money, the class that became CalendarDate and some nice unit tests for TimeInterval. Benny Sadeh contributed the initial version of BusinessCalendar and TimeUnit, and did some refactoring on the code base. Vlad Gitlevich contributed the SerializationTest. Vlad Gitlevich and John Corbett each acted as programming pairs. Kai Yu identified and fixed key bugs. Michael Two provided valuable debugging assistance. Alan Lovejoy helped clarify the TimePoint/Duration/Interval distinction.
